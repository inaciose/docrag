# docrag

#docRAG
=======

A local-first document ingestion and RAG pipeline for:

- PDFs
- scanned documents
- photos
- datasheets
- manuals
- mixed text/image documents

Pipeline:

PDF/IMG
→ OCR
→ OCR cleanup
→ Vision-Language extraction
→ Chunking
→ Embeddings
→ Qdrant
→ Retrieval-Augmented Generation
